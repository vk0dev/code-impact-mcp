## Snapshot

- CodeImpact MCP: fast local pre-commit gate, PASS/WARN/BLOCK verdict, MIT, zero setup
- CodeGraphContext: broader context retrieval and repository reasoning
- Depwire: broader dependency analysis platform with heavier setup and stored analysis
- code-graph-mcp / RepoGraph / code-pathfinder: stronger for exploration and navigation than for a single verdict gate
